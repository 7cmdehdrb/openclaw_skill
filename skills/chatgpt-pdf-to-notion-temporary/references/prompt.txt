다음 스키마로 논문 요약을 작성한다 (로컬 PDF 본문 추출 기반):

# 0) 논문 정보
- 출간 연도
- 출간 저널
- citation 수
- `### APA` (또는 **APA**) 다음 줄에 citation 본문을 인용(quote)으로 기록
- `### BibTeX` (또는 **BibTeX**) 다음 줄에 BibTeX 본문을 인용(quote)으로 기록

# 1) 문제 상황
- 연구가 해결하려는 핵심 문제
- 기존 방법의 한계
- 왜 이 문제가 중요한지

# 2) Proposed Method (제안 방법)
- 핵심 아이디어
- 시스템/알고리즘 구성
- 중요한 설계 변수 또는 수식/기호

# 3) 정량 및 정성적 결과
## 정량 결과
- 수치 지표, 비교 결과, 성능 변화
- 가능하면 표로 정리

## 정성 결과
- 동작 특성, 장단점, 데모/사례

## 한줄 평가
- 논문의 실질 기여를 1~2문장으로 요약

출력 형식 요구:
- 마크다운 구조 유지 (`#`, `##`, `###`)
- 불릿/표 적극 사용 가능
- 수식/기호(예: Δ, τ, H∞) 보존
- 본문에 없는 내용은 추정하지 말 것
- 불명확한 값은 `본문 확인 필요`로 표시
