# Output Format (Notion Body)

Use this structure in order.

## 1) 한 줄 요약
- `- <회의 전체를 한 문장으로 요약>`

## 2) 회의 핵심 요약
- `- 핵심 포인트 1`
- `- 핵심 포인트 2`
- `- 핵심 포인트 3`

## 3) 상세 논의 내용
### 주제 A
- `- 논의 내용`
- `- 근거/수치`

### 주제 B
- `- 논의 내용`

## 4) 결정 사항
- `- [결정] <결정 내용>`
- `- [결정] <결정 내용>`

## 5) 할 일 (Action Items)
- `- [ ] <작업명> | 담당: <이름/미지정> | 기한: <날짜/미정> | 우선순위: <높음/중간/낮음>`
  - `- 메모: <맥락 한 줄>`

## 6) 오픈 이슈 / 확인 필요
- `- <질문 또는 미확정 항목>`
- `- <해석 필요 문장>`

## 7) 메타
- `- 원본 파일: <filename.txt>`
- `- 정리 시각: <YYYY-MM-DD HH:MM AM/PM>`
