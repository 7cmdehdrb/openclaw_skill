# 1) 문제 상황
- 소프트 로봇 핸드는 순응성 덕분에 다양한 물체 파지에 유리하지만, 파지 안정성(미끄럼/불안정) 추정을 위한 대규모 데이터 수집 비용이 크다.
- 물체별 형상/재질 다양성과 동적 변형(예: 잡을 때 형상 변화) 때문에 안정성 판별 일반화가 어렵다.

# 2) Proposed Method (제안 방법)
- 손가락에 **유연(force) 센서 어레이**를 통합한 anthropomorphic 소프트 핸드를 개발하고, 시계열 촉각 신호로 파지 상태를 추정한다.
- 실험 프로토콜:
  - 물체를 테이블에서 **1.5 s** 유지 후 **1 s** 내 들어올림
  - 공압 **50 / 80 / 120 kPa**에서 반복 실험
  - 총 **20개 물체**, **300회 grasp trial**(각 trial의 force 시계열) 수집
- 데이터 효율을 위해 **OSL(one-shot learning) + LSTM** 기반 3개 네트워크(LSTM, Siamese LSTM, Triplet LSTM)를 비교한다.

# 3) 정량 및 정성적 결과
## 정량 결과
| 항목 | 본문에서 확인된 수치 |
|---|---|
| Triplet LSTM 안정성 추정 정확도 | **89.96%** |
| LSTM 안정성 추정 정확도 | **88.00%** |
| Siamese LSTM 안정성 추정 정확도 | **85.16%** |
| 센서 샘플링 주파수 | **100 Hz** |
| FFA 최대 저항 grip force (120 kPa) | **6.05 ± 0.63 N** |
| 손 수직 파지력(Ø50 mm cylinder, 120 kPa) | **26.53 ± 0.87 N** |

## 정성 결과
- 낮은 압력(50 kPa)에서는 일부 물체에서 미끄럼/불안정이 증가했고, 물체 특성(변형성, 형상)에 따라 성능 편차가 나타났다.
- Triplet 구조가 hardest negative를 활용해 구분력을 높여 가장 높은 평균 정확도를 보였다.
- 제안 시스템은 센서 기반 파지 안정성 피드백을 통해 물체별 필요 압력을 선택하는 제어 확장 가능성을 제시한다.

## 한줄 평가
센서 통합 소프트 핸드와 OSL-LSTM 기반 안정성 추정기를 결합해, 제한 데이터 조건에서도 **85.16–89.96%** 수준의 파지 안정성 추정을 달성한 실험 중심 연구다.
