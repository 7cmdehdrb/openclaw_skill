# 1) 문제 상황
- 소프트 로봇의 상태 추정에서 강한 비선형성과 불확실성 때문에 기존 EKF 계열의 성능 저하가 발생한다.
- 모델 미스매치/외란 환경에서 추정 오차를 억제할 수 있는 강건한 상태추정기가 필요하다.

# 2) Proposed Method
- 논문은 H-infinity 기반 확장 칼만 필터(EKF 변형)를 제안해 비선형 소프트 로봇 시스템 상태를 추정한다.
- 표준 EKF 대비 worst-case 관점의 강건성 보강을 목표로 하며, 모델 불확실성과 잡음 영향 완화에 초점을 둔다.

# 3) 정량 및 정성적 결과
## 정량 결과
- 본문에서 제시된 비교 지표는 원문 세부 수치 기준으로 확인 필요(본문 확인 필요).

## 정성 결과
- 제안 추정기는 외란/모델 불일치 조건에서 안정적인 추정 성능을 보였다고 보고한다.
- 소프트 로봇의 고비선형 상태 추정에서 강건 추정 프레임워크의 실용성을 제시한다.

## 한줄 평가
- H∞ 기반 EKF를 통해 소프트 로봇 상태추정의 강건성을 강화하려는 접근을 제시한 논문이다.
