# 1) 문제 상황
- 소프트 로봇은 비정형 환경 적응성과 안전한 물리 상호작용 측면에서 강점이 있지만, **강한 비선형성, 히스테리시스, 연속체 구조로 인한 사실상 무한 자유도(DoF)** 때문에 실사용 모델링/제어가 어렵다.
- 물리 기반 모델(PCC, Cosserat rod 등)은 해석 가능성과 구조적 일관성은 높지만, 재료 비선형성과 환경 불확실성을 완전히 반영하기 어렵다.
- 데이터 기반 모델은 비선형 거동 표현력은 높으나 대규모 학습 데이터 의존성과 일반화 저하 문제가 크다.

# 2) Proposed Method (제안 방법)
- 본 논문은 2020–2025년 약 60편을 대상으로, 소프트 로보틱스 연구를 **시스템 모델링**과 **제어**의 두 축으로 재분류/비교한 리뷰다.
- 시스템 모델링은 다음 3계열로 정리한다.
  - 물리 기반(physics-based)
  - 데이터 기반(data-driven)
  - 하이브리드(hybrid: 물리 모델 backbone + 학습 기반 보정)
- 제어는 다음 4계열로 정리한다.
  - 전통 제어(PID/MPC 등)
  - 데이터 기반 제어
  - 강화학습(RL) 기반 제어
  - 하이브리드 제어
- 핵심 관점은 **물리적 해석 가능성(interpretability)** 과 **비선형 표현/적응력** 사이의 trade-off이며, 최근에는 이를 결합하는 하이브리드 전략이 주류로 이동했다고 분석한다.

# 3) 정량 및 정성적 결과
## 정량 결과
| 항목 | 본문에서 확인된 수치 |
|---|---|
| 분석 범위 | 2020–2025년 약 60편 |
| 데이터 기반 모델 데이터 규모 예시 | 약 20,000–54,000 samples |
| RNN 기반 상태추정 예시 (비접촉) | RMSE: 길이 1.10±0.02 mm, pitch 0.62±0.05°, roll 0.71±0.06° |
| LSTM 기반 self-sensing 예시 (접촉) | 평균 굽힘각 오차 2.36–3.95°, 평균 접촉력 오차 약 0.002–0.01 N |
| 비전 기반 CNN 예시 (내부 카메라) | RMS 오차: α 1.03±0.03°, β 1.39±0.10° |
| 하이브리드 모델(잔차학습) 예시 | EEF 위치/속도/가속도 RMSE: 2.2 mm, 1.1 mm/s, 1.1 mm/s²; 물리모델 단독 대비 오차 약 66–70% 감소 |
| 하이브리드 제어 예시 | 궤적 추종 오차 2.295 mm |
| 제어 주파수 경향(논문 내 비교) | RL 기반 평균 약 150 Hz, 하이브리드 제어 전체 루프 평균 약 30 Hz (단일 루프 계산속도는 약 300 Hz 수준 보고) |

## 정성 결과
- **시스템 모델링 측면**: 물리 기반 → 데이터 기반 → 하이브리드로 진화하며, 최근에는 하이브리드가 정확도/일반화/해석가능성 균형에서 SOTA 경향을 보인다.
- **제어 측면**: 하이브리드 제어는 안정성과 실용성이 높고, RL 제어는 표현력/적응력이 커서 점차 경쟁력 있는 대안으로 부상한다.
- **남은 과제**: RL의 Sim-to-Real gap, 모델 정의(저차원 표현) 자체의 타당성 검토, 방법론 표준화가 핵심 미래 과제로 제시된다.

## 한줄 평가
이 리뷰는 소프트 로보틱스의 모델링·제어를 학습 통합 수준 관점에서 재구성하여, **하이브리드 접근의 실질적 우위와 RL의 잠재력/한계(Sim-to-Real)** 를 근거 중심으로 정리한 로드맵 성격의 논문이다.
