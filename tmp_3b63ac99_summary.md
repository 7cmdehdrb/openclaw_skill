# 1) 문제 상황
- 연속체/소프트 매니퓰레이터는 고유한 순응성과 안전성 장점이 있지만, 제어 관점에서 **통일된 설계·해석·제어 프레임워크가 아직 확립되지 않았다**고 논문이 지적한다.
- 핵심 난점은 (i) 사실상 무한 자유도(연속 변형), (ii) 재료 비선형성/히스테리시스, (iii) 로봇마다 상이한 설계·구동 구조로 인한 일반화 어려움이다.
- 따라서 실사용을 위해서는 빠르고 신뢰성 있는 실시간 제어기가 필요하지만, 모델링 난이도와 계산비용이 큰 병목으로 제시된다.

# 2) Proposed Method (제안 방법)
- 본 논문은 새로운 단일 알고리즘 제안이 아니라, 연속체/소프트 매니퓰레이터 제어 전략을 체계적으로 정리한 **서베이**다.
- 제어기를 다음 축으로 분류해 비교한다.
  - **Model-based static(kinematic) control**
  - **Model-free static control**
  - **Model-based dynamic control**
  - **Model-free dynamic control**
- 분류 기준은 제어 목표 공간(작업공간/구성공간), 모델 의존성, 실시간성/계산복잡도, 불확실성 대응 성격이며, Table 1 중심으로 대표 연구를 정리한다.

## 정성 결과
- **정적/기구학 제어**는 구현이 비교적 단순하고 다수 사례가 축적되어 있으나, 고속/동적 과제에서는 한계가 있다.
- **동적 제어**는 산업 적용에 중요하지만 모델 정확도·계산비용·실시간 구현 부담이 크다.
- **모델 프리 접근(학습 기반 포함)**은 모델링 부담을 줄이고 복잡 비선형 대응 잠재력이 있으나, 학습 안정성·일반화·저수준 제어 설계 이슈가 남아 있다.
- 논문은 장기적으로 모델 기반과 학습 기반을 결합한 하이브리드 방향, 저수준 제어 및 하드웨어-제어 공동설계 중요성을 강조한다.

## 한줄 평가
이 서베이는 소프트 매니퓰레이터 제어를 모델 기반/프리 × 정적/동적으로 구조화해, 각 접근의 실무적 trade-off(해석성·실시간성·일반화)를 명확히 정리한 기준 문헌이다.
