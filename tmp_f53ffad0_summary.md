# 1) 문제 상황
- 소프트 연속체 매니퓰레이터는 고비선형 동역학/기구학 특성 때문에 상태 추정이 어렵고, 센서를 많이 붙이면 로봇의 본래 유연성을 해칠 수 있다.
- 따라서 제한된 센서 조건에서 관측기/필터 기반 상태 추정이 필요하지만, 기존 EKF는 모델 불확실성이 큰 소프트 로봇에서 성능 저하가 발생할 수 있다.

# 2) Proposed Method (제안 방법)
- 논문은 고비선형 소프트 로봇 상태추정을 위해 **H∞ 기반 Extended Kalman Filter (H∞-EKF)** 를 제안한다.
- 단일 섹션 3개 pMA(공압근육) 기반 소프트 연속체 매니퓰레이터 모델을 상태공간 형태로 정식화하고,
  - pMA elongation
  - elongation rate
  - task-space coordinates
  추정을 수행한다.
- 제안 필터는 실험 검증된 소프트 매니퓰레이터 모델 시뮬레이션에서 성능을 평가한다.

# 3) 정량 및 정성적 결과
## 정량 결과
| 항목 | 본문에서 확인된 수치 |
|---|---|
| 논문/학회 | 2019 American Control Conference (ACC) |
| 문서 분량 | 7 pages |
| pMA elongation 추정 오차 | **2 × 10⁻⁴ m 이하**(near-perfect tracking으로 보고) |
| elongation rate 추정 특성 | 오차가 **약 0.02** 근방으로 수렴(0으로 완전 수렴하지 않음) |
| 필터 샘플링 스텝 관련 언급 | **0.001 s**에서 상태 발산 가능성 언급(고비선형성 영향) |

## 정성 결과
- 제안 H∞-EKF는 pMA 신장량과 작업공간 좌표 추정에서 정확한 추정 성능을 보였다.
- 반면 신장속도 추정은 모델 불확실성 영향으로 상대적으로 성능이 낮다고 보고했다.
- 논문은 소프트 로봇의 강한 비선형/불확실성 환경에서 H∞ 설계가 견고한 추정에 유리함을 강조한다.

## 한줄 평가
센서 최소화가 필요한 고비선형 소프트 연속체 로봇에 대해, H∞-EKF로 신장량/작업공간 상태를 안정적으로 추정하는 가능성을 보인 상태추정 연구다.
