# 1) 문제 상황
- 미지 물체(unseen objects) 파지에서는 지각/동작 불확실성이 크며, 기존 모델 기반 파지는 물체/핸드/접촉 모델의 사전지식에 의존한다.
- 소프트 핸드는 순응성(compliance)으로 불확실성에 강하지만, 손가락 변형이 예측 어려워 전통적 접촉점 기반 파지계획을 직접 적용하기 어렵다.

# 2) Proposed Method (제안 방법)
- 부분 포인트클라우드 입력으로 파지 성공 가능성을 예측하는 **3D CNN 기반 파지 선택**을 제안한다.
- 2D/2.5D 이미지 기반 및 고정 top-grasp 중심 접근과 달리, **top/side grasping direction + wrist orientation**을 함께 고려한다.
- 시스템 구성:
  - Baxter 양팔 end-effector에 소프트 핸드 장착
  - 상체 depth sensor로 테이블 위 물체의 부분 포인트클라우드 획득
  - 후보 grasp에 대해 3D CNN이 성공 likelihood를 추정해 파지 자세 선택

# 3) 정량 및 정성적 결과
## 정량 결과
| 항목 | 본문에서 확인된 수치 |
|---|---|
| 실제 로봇 파지 성공률(미지 물체) | **87%** (3DCNN) |
| 비교기준 RAND | 약 **15%** |
| 비교기준 PCA / SVM | 각각 약 **62%** |
| 학습/평가 물체 수 | 학습 10개, 테스트 10개 |
| 성공 판정 기준 | 파지 후 **3초** 유지 시 성공 |

## 정성 결과
- 3D CNN 기반 접근은 소프트 핸드의 순응성과 결합되어, 미지 물체에서도 랜덤/기준 접근 대비 높은 강건성을 보였다.
- 논문은 노이즈/가림(occlusion) 조건에서도 예측 성능 비교를 제시하며, geometry-aware한 3D 표현이 유리함을 강조한다.
- 다만 일부 실패 사례도 존재하며, 물체 형상/접촉 조건에 따른 한계가 보고된다.

## 한줄 평가
사전 물체 모델 없이 부분 포인트클라우드만으로 소프트 핸드 파지를 학습하는 3D CNN 프레임워크를 제시했고, 실제 실험에서 **87% 성공률**로 실용성을 입증했다.
